# ADYC

System to manage extracurricular classes for high school students at CECyTEG Cortazar, Guanajuato, Mex.

### Technologies stack!
 * Web API
* Identity Framework
* ASP.MVC 4
* Entity Framework 6
* Autofact
* TDD
* NUnit
* Moq
* jQuery
* DataTables
* Twitter Bootstrap
* Admin LTE
* SQL Server
